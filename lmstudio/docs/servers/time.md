# time

**MCP server:** `time`  
**Package:** `mcp-server-time`

Current time and timezone conversion.

---

## When to use

| User asks | Tool |
|---|---|
| “What time is it?” | Current time in default timezone |
| “What time is 3pm EST in Tokyo?” | Convert between timezones |
| Scheduling context | Anchor “now” for relative dates |

No dependencies on other servers.
